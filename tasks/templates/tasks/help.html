About

This is a task management application built with Django; that focuses on creating tasks with prerequirements that ensure the user can navigate the intricacies of their objectives with subtasks of various requirements, that vary from all, one, or optional.

The name is an attempt to imitate the naming conventions of the Imperium of Man from Warhammer 40k, where Astra and Agenda are latin for "stars" and Agenda means "list" (according to google translate) respectively.

Help

First, register an account or log in if you already have one.

After logging in, you can add tasks using the Add Task which is the floating + button. Each task can have a title, description, and can be marked as completed or in progress.

Dependencies

Dependencies are the core of this application. Upon creating a main task, if you go to the detail pages via the detail button found after uncollapsing a task, you can add dependencies to that task.

Where functionally it is similar to adding a task, but you are greeted with the dependency which dictates the type of prerequisite:

  • ALL (AND): All tasks mark AND must be completed before the main task can be marked as completed.
  • ONE (OR): At least one of the designated OR tasks must be completed before the main task can be marked as completed.
  • OPT (Optional): The all OR tasks are optional and have no bearing.

The application enables mixed prerequirements allowing for complex task management.

Additionally you can create subtasks for the subtasks up to a limit of two levels deep.

  • ↳ Subtask (Level 1)
  • ↳ Sub-subtask (Level 2)
